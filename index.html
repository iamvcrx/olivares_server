Politique de confidentialité

Cette application WhatsApp collecte des messages vocaux envoyés par les utilisateurs qui seront utilisés pour utiliser une intelligence artificielle strictement dans le cadre d'un projet de production artistique.

Les données collectées incluent : le numéro de téléphone de l'utilisateur, son prénom/nom s'il est fourni, et le fichier audio du message vocal.

Ces données ne sont pas partagées avec des tiers et sont uniquement utilisées dans le cadre du projet de Clara Olivares.

Pour toute question, contactez-nous à bigand@ircam.fr.
